# Design Notes

## Fonts
- Chillax: Bold/Semibold/Regular (headings, brand text)
- Satoshi: Bold/Medium/Regular (body, UI text)

## Colors (from splash)
- Deep BG: #0D0221, #1A0D3D, #1A0533, #1A0B2E
- Purple: #3D1480, #6443F4
- Pink: #F94498
- Orange: #FF9327
- Yellow: #FDCD0A
- Surface: #24103E
- Border: rgba(123,68,230,0.25)
- Text Primary: #FFFFFF
- Text Secondary: #D3CFD8
- Text Tertiary: #A79FB2

## Assets
- mascot-dark.png (duck mascot for dark bg)
- logotype-dark.webp (TRAVI logotype for dark bg)
- Destination photos from Unsplash: bali, santorini, kyoto, paris, dubai, tokyo, barcelona

## Design Principles
- No emojis, use FontAwesome icons
- Chillax for headings, Satoshi for body
- Gradient CTA: #6443F4 → #F94498
- Glassmorphism cards with subtle borders
- Deep purple backgrounds with depth layers
